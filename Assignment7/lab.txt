Commands and their relationships:
open- opens the library code
read- reads the data
close- closes the library code (has to be paired with an open)
fstat- to get the size of library and allocate memory
mmap- creates a mapping between the library and the .text and .data in memory
mprotect- to protect the mapping from attacks
